## التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- pnpm 10+

### خطوات التثبيت

```bash
# الانتقال إلى مجلد المشروع
cd zad_albaheth

# تثبيت المتطلبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# بناء المشروع للإنتاج
pnpm build

# تشغيل المشروع المبني
pnpm start
```
